# 🤖 Build with AI

Scaffold-ETH 2 is designed to work great with AI coding agents like [Claude Code](https://claude.ai/code), [Cursor](https://cursor.sh), [Cline](https://github.com/cline/cline), and others.

Every SE-2 project ships with an **AGENTS.md** file that gives AI agents deep context about the project structure, available commands, architecture patterns, and code style. On top of that, **Skills** provide reusable instructions that agents can follow to add features like NFT contracts, Sign-In with Ethereum, subgraph indexing, and more.

## What's included

:::note[📄 AGENTS.md — Project Context for AI Agents]
Every SE-2 project includes an `AGENTS.md` file at the root. AI agents automatically read this file when they enter your project, giving them full context about the architecture, commands, and patterns.

[Learn more about AGENTS.md →](/build-with-ai/agents-md)
:::

:::note[⚡ Skills — Reusable Agent Instructions]
Skills are markdown files that teach AI agents how to add specific features to your SE-2 project. Instead of explaining everything from scratch, just point your agent to a skill.

[Browse available Skills →](/build-with-ai/skills)
:::

## Quick Start

Open your AI agent and give it the orchestrator skill URL along with your dApp idea:

```
https://docs.scaffoldeth.io/SKILL.md Build me a mass payout dApp that
lets a user upload a CSV of addresses and amounts, then sends ETH to
all of them in a single transaction using a batch-transfer contract.
```

That's it. The orchestrator skill tells your agent how to scaffold a new SE-2 project with `npx create-eth@latest`, walks it through the full setup, and based on what you asked for, the agent picks the right sub-skills from `.agents/skills/` to implement the features correctly.

If your prompt mentions indexing, the agent reads the **ponder** skill. NFTs? It grabs **erc-721**. Payments? **x402**. The skills ship with every SE-2 project — the agent finds and uses them automatically.

### Already have a project?

If you've already scaffolded, just point your agent to a skill directly:

```
Read .agents/skills/erc-721/SKILL.md and add an NFT contract
to this project.
```

The agent picks up `AGENTS.md` for project context and follows the skill's patterns for implementation.
